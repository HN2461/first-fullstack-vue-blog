# Python 零基础入门 01：Python 是什么

Python 是一种编程语言。

你可以把编程语言理解成：**人和电脑沟通时使用的一套指令格式。**

人平时说中文、英文。电脑不能直接理解这些自然语言，所以我们用 Python 这样的编程语言，把想让电脑做的事情写清楚。

## Python 能做什么

Python 可以做很多事情：

- 自动整理文件
- 处理 Excel 表格
- 写网站后端
- 做数据分析
- 写爬虫采集公开数据
- 做人工智能相关开发
- 写一些日常小工具

对零基础学习者来说，不需要一开始就选方向。先把基础语法学会，后面再决定往哪个方向深入。

## 为什么 Python 适合入门

Python 的优点是：代码看起来比较接近日常表达。

比如让电脑显示一句话：

```python
print("你好，Python")
```

你大概能猜到：`print` 是“打印、输出”的意思，括号里是要显示的内容。

如果换成一些更底层的语言，同样的事情可能需要写更多代码。Python 对新手更友好。

## Python 不是万能的

Python 很适合入门，也很实用，但它不是所有场景的最佳选择。

比如：

- 写大型高性能游戏引擎，通常不会首选 Python。
- 写手机原生 App，通常会用其他语言。
- 写浏览器里的交互页面，通常会用 JavaScript。

但是，Python 的应用范围已经足够广，特别适合把编程思维先建立起来。

## 第一个感受：命令从上往下执行

看下面的代码：

```python
print("第一行")
print("第二行")
print("第三行")
```

运行结果：

```text
第一行
第二行
第三行
```

Python 默认会从上往下执行代码。

这就像你写了一张任务清单：

1. 先显示“第一行”
2. 再显示“第二行”
3. 最后显示“第三行”

电脑会按照顺序做。

## 代码里的符号要用英文符号

新手很容易把中文标点写进代码里。

正确写法：

```python
print("你好")
```

错误写法：

```python
print（"你好"）
```

第二段里括号是中文全角括号，Python 看不懂。

写代码时，括号、引号、逗号，建议都使用英文输入法下的符号。

## 本篇小练习

复制下面代码运行：

```python
print("我正在学习 Python")
print("Python 可以帮我让电脑自动做事")
```

运行结果：

```text
我正在学习 Python
Python 可以帮我让电脑自动做事
```

然后把文字改成你自己的话，再运行一次。

## 本篇小结

- Python 是一种让人和电脑沟通的编程语言。
- Python 语法相对简单，适合零基础入门。
- Python 代码默认从上往下执行。
- 写代码时尽量使用英文输入法的符号。
