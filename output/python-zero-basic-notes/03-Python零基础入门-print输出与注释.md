# Python 零基础入门 03：print 输出与注释

`print` 是 Python 新手最先接触的工具。

它的作用很简单：**把内容显示到屏幕上。**

## 输出一句话

```python
print("你好，Python")
```

运行结果：

```text
你好，Python
```

括号里的内容会被显示出来。

## 输出多句话

```python
print("今天学习 Python")
print("第一步是学会输出内容")
print("慢慢来，不着急")
```

运行结果：

```text
今天学习 Python
第一步是学会输出内容
慢慢来，不着急
```

每写一个 `print`，默认会换一行。

## 输出多个值

`print` 可以一次输出多个内容，中间用英文逗号隔开。

```python
print("姓名：", "小明")
print("年龄：", 18)
```

运行结果：

```text
姓名： 小明
年龄： 18
```

注意，逗号要用英文逗号 `,`。

## 输出空行

如果想输出一个空行，可以这样写：

```python
print("第一段")
print()
print("第二段")
```

运行结果：

```text
第一段

第二段
```

中间空了一行。

## 什么是注释

注释是写给人看的说明，Python 不会执行它。

在 Python 里，单行注释用 `#` 开头。

```python
# 下面这行代码会输出一句问候
print("你好")
```

运行结果：

```text
你好
```

第一行不会被执行，因为它是注释。

## 注释有什么用

注释适合用来解释“为什么这样写”。

比如：

```python
# 年龄先写死，后面学 input 后再改成用户输入
age = 18
print(age)
```

这里的注释告诉读者：现在写 `18` 只是临时做法。

## 不好的注释

下面这种注释意义不大：

```python
# 打印你好
print("你好")
```

因为代码本身已经很清楚。

新手阶段可以多写一点注释帮助自己理解，但以后要逐渐学会写有价值的注释。

## 临时关闭一行代码

注释还有一个常见用途：临时让某行代码不执行。

```python
print("第一行")
# print("这一行暂时不运行")
print("第三行")
```

运行结果：

```text
第一行
第三行
```

## 本篇小练习

复制并运行：

```python
# 这是我的个人信息练习
print("姓名：", "小明")
print("城市：", "北京")
print()
print("学习目标：学会 Python 基础")
```

运行结果：

```text
姓名： 小明
城市： 北京

学习目标：学会 Python 基础
```

然后把姓名、城市、目标改成你自己的内容。

## 本篇小结

- `print` 用来把内容显示到屏幕上。
- 多个 `print` 默认会分多行输出。
- `print()` 可以输出空行。
- `#` 后面的内容是注释，Python 不会执行。
- 好注释主要解释原因，而不是重复代码表面意思。
