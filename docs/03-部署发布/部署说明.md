# 部署说明

本项目已经拆分为两个独立应用：

- `frontend/`：前端 Vue + Vite
- `backend/`：后端 Express + MongoDB

## 前端

```bash
cd frontend
npm install
npm run build
```

构建产物在 `frontend/dist`。

## 后端

```bash
cd backend
npm install
npm run create:admin
npm run start
```

生产环境建议用 PM2：

```bash
cd backend
pm2 start ecosystem.config.cjs
```

## 数据库

MongoDB 独立运行，连接串在 `backend/.env` 中配置。

## 访问路径

- 前端：Nginx 静态托管 `frontend/dist`
- API：Nginx 反向代理 `/api` 到 `127.0.0.1:3001`
- 上传文件：Nginx 反向代理 `/uploads` 到 `127.0.0.1:3001`
