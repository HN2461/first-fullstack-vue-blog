# 个人全栈博客系统

本仓库已拆分为两个独立应用：

- `frontend/`：前端 Vue 3 + Vite
- `backend/`：后端 Express + MongoDB

## 启动方式

### 前端

```bash
cd frontend
npm install
npm run dev
```

### 后端

```bash
cd backend
npm install
npm run dev
```

## 文档入口

先看 [docs/README.md](docs/README.md)，里面有中文分类索引。

## 目录说明

- `frontend/`：前端应用
- `backend/`：后端 API 服务
- `deploy/`：Nginx / PM2 部署模板
- `docs/`：部署与发布说明
- `uploads/`：后端上传目录
